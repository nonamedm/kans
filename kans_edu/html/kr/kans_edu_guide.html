• 안녕하세요?
    교재의 진행을 도와줄
    안지’ 라고 해요.
    만나서 반갑습니다.

  • 교재의 진행에 앞서
    교재 사용 방법에 대해
    간단히 설명 드리고 넘어갈게요!

  • 먼저 목차.
    교재의 전체 목록을 볼 수 있는 곳이에요.
    클릭하면...

  • 이렇게
    원하는 챕터로 이동 할 수 있는
    메뉴가 나타난답니다.

  • 목차를 닫고 싶을때는
    메뉴 상단의 X버튼이나
    목차 버튼을 클릭하시면 됩니다.

  • 목차를 열지 않아도
    상단의 빠른탭을 클릭하시면
    각 챕터별로 바로 이동 하실 수 있습니다.

  • 교재 본문의 이미지는
    클릭하면 큰 이미지로 보실 수 있습니다.

  • 이미지를 자세히 보시려면
    이렇게 큰 이미지로 확인하시면 되요.

  • 큰 이미지에서 나가실때는
    우측 상단의 X버튼이나
    이미지 중앙을 클릭하시면 됩니다.

  • 이전, 다음 페이지로의 이동은
    교재 우측 하단의 버튼을 이용 하시면 됩니다.
    현재 페이지 / 전체 페이지로 표시되며
    교재를 닫을때는 나가기 버튼을 눌러주세요.

  • 교재를 초기화 하시려고 할때는
    좌측 상단의 로고를 클릭해 주세요!

  • 그럼 ‘방사선과 안전규제의 만남
    교육을 시작해 볼까요?
